# WebChat

WebChat eshte nje platforme web per komunikim pothuajse ne kohe reale qe mundeson regjistrimin dhe identifikimin e perdoruesve, dergimin dhe marrjen e mesazheve, bisedat private dhe ne grup, si dhe ruajtjen e historikut te komunikimit.

## Karakteristikat Kryesore

### Menaxhimi i Perdoruesve

* Regjistrimi i perdoruesve
* Hyrja ne sistem (Login)
* Dalja nga sistemi (Logout)
* Menaxhimi i sesioneve

### Komunikimi

* Biseda private
* Biseda ne grup
* Perditesimi i mesazheve me AJAX
* Historiku i bisedave

### Menaxhimi i Projektit

* GitHub Issues
* GitHub Milestones
* GitHub Projects
* Pull Requests
* Branch Management
